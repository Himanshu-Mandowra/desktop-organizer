# DeskCanvas

A customizable desktop wallpaper organizer and workspace layout editor built with React and React-Konva.

## Features

* Upload custom wallpapers
* Create draggable workspace sections
* Resize sections dynamically
* Snap-to-grid alignment
* Modern glassmorphism UI
* Sidebar editor for section customization
* Organize desktop folders visually
* Export clean desktop-ready wallpapers
* Persistent layouts using localStorage

## Tech Stack

* React
* React-Konva
* Tailwind CSS
* Lucide React

## Installation

```bash
git clone <repo-url>

cd <file>

npm install

npm start
```

## Usage

* Upload a wallpaper
* Add sections for categories like:

  * Frontend
  * Backend
  * Node.js
  * MongoDB
  * SQL
* Drag and resize sections
* Customize colors and titles
* Export the final wallpaper
